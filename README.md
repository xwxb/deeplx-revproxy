

简易 Deeplx 免费api 中转项目，统一管理多个端点，请求失败则自动降级    

### Usage

重命名 `config.yaml.tmpl` 填入自己要管理的端点，`go run .` 启动程序即可

### Feature

- 兼容原有 api
